# ERFNet

# Prerequisites
    Python 3
    NVIDIA GPU + CUDA cuDNN
    PyTorch 
# Run
1.train the model
    python train.py
2.test the model
    python test.py

# Download Datasets
    We use FFHQ, CelebA-HQ, Dunhuang Mogao Grottoes Mural and Paris Street-View datasets. Liu et al. provides 12k irregular masks as the mask.
